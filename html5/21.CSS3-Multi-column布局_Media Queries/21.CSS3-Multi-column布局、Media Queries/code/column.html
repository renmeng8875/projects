当地时间20日零时30分（北京时间14时30分）左右，美国科罗拉多州首府丹佛市的一家电影院发生一起枪击惨案，当很多观众正在这家影院熬夜观看《蝙蝠侠：黑暗骑士崛起》的首映时， 一名戴着防毒面具、拿着重武器的枪手偷偷闯了进来。这名24岁的男子先抛“催泪弹”，后拿起突击步枪等武器向观众胡乱射击，造成重大伤亡，至少12人遇难，50人受伤。 电影院内忽然烟雾弥漫，观影的人群还不明白发生了什么事。有些人甚至以为这是电影首映式专门安排的一个环节。 观众之一的查伊尔·杰克逊回忆说：“电影的第一个动作场景刚开始，我右边就出现了一个全身黑衣的男子，头戴黑色面具。一开始，我还以为这是电影首映式的一个环节。他朝人群扔了催泪弹，之后人群陷入了慌乱。他掏出一把枪就开始扫射。我一下子没明白过来，这到底是真实场景还是在做梦。” 19岁的萨莉娜·乔丹称，她看到一个女孩脸上中枪，还有一个大概9岁的女孩肚子中了一枪。 目击者克莱斯特·琼斯告诉一家电视台，电影开场20多分钟后，枪手就冲到了屏幕前面，“我们那时刚看了20或30分钟电影吧，先是看到了一阵烟雾，都以为是放烟花或者什么的，但是之后就看到有人倒下了，然后就是不间断的枪声”。 目击者詹妮弗·西格透露说：“我告诉我的朋友，‘我们要赶快出去’，那时，枪手就开始射击那些逃向出口的人了。” “我看到一个男孩在地上爬，看到一个女孩鲜血直涌，看到一些人的背上、胳膊上满是弹孔。我还看到一个男的身上的短裤都被脱下来了，好像背上还是哪里中枪了。”侥幸死里逃生的多诺万·塔特告诉媒体。（高美） 【亲历2】 小女孩躺在地上鲜血直流 目击者莎依拉·罗德描述说，她看到一个十来岁的小女孩，躺在地上，鲜血直流。“她眼神里充满恐惧……我们互相对视了一眼，我能看出来，她快不行了。” 海登·米勒当时在8号影厅，他说开始他还以为是鞭炮在响，直到有人跑进来，大喊：“那边有人开枪杀人啦！”海登听到了几声枪声，“就像是小鞭炮在响，之后我们就听到人们尖叫起来，人们全都蜷着身子向电影院外面跑去。” “他们站起来，就向紧急出口跑去，当她回头来看，就只看到枪手慢慢地走上台阶，漫无目的地对着人群随意扫射。”一位男子和现场逃生的一对情侣聊天时，这样讲述他们的经历。 “扫射一直在持续，之后我们就听不到什么声音了。我们最后终于跑出来，看到有人在流血，有人显然已经死了，到处都是一片混乱。”这对情侣说。 有目击者表示：“我从没有那么害怕过。我们那时全都困在电影院里，绝望无助，根本出不去。”（高美） 我和米歇尔（总统夫人）对此次可怕的枪击事件感到非常震惊和悲痛。我们必须团结一致如同一家人，共同面对如此黑暗、艰难的时刻。——奥巴马
